Installation Instructions

Tip!

Depending on your system security you may be warned by Windows SmartScreen, or other such similar security system, that OverTrack setup has been prevented from running - in the case of Windows SmartScreen, click on More info, then, Run anyway to proceed. This only needs to be done once..

  • 1. Setup
  • 2. Link Account
  • 3. General Use

Step through the install process. We reccomend leaving "start on login" enabled so you don't miss any games.

Upon first launch, OverTrack will need to fetch additional requirements.
This process can take some time on first run, but OverTrack will reuse most of the downloaded requirements for updates.

Upon first launch of the client you will be asked to authenticate/pair the client to your account using the OverTrack site.

If you are not logged in you will instead be presented with the welcome page - simply use the sign-in button and you will be redirected to the authenticate page.

OverTrack has a taskbar icon showing the current state and will present toasts updating you to tracking events (toasts may not be visible in fullscreen) .



To ensure SR is tracked correctly you must visit the Role Selection Menu BEFORE and AFTER each game

For best results for tracking scores, let the tracker see all score screens - this means not tabbing out or leaving the game at the round/game end score screens.

To close the client, right click the tray icon and select Quit

Notes:

Fullscreen tracking is not available in Windows 7.

In order to help with bugs and development the client will upload logs, extracted data, and occasionally full screenshots.
Care is taken in the application to mask out any non-game windows or content before extrating data or saving screenshots.
